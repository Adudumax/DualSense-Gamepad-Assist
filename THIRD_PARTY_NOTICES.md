## Gyrosteer Reference

The CSP DualSense output approach was informed by the user-provided Gyrosteer
script by Dmitrii Alekseev. This fork does not include gyroscope steering and
does not read gyroscope or accelerometer data.

## Forza Horizon DualSense Reference

The user-provided Forza Horizon DualSense Python script informed the priority
ordering and timing approach used for adaptive-trigger feedback. This fork
contains a CSP Lua implementation written for Assetto Corsa.
